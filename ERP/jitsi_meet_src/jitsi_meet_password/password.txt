SEND PASSWORD FROM ODOO TO JITSI-MEET

STEPS

1. ENABLE PASSWORD AND USER VIDEO MEETING FORM BY XML

   views/jitsi_meet_views.xml
   
2. READ PASSWORD TO DATA FIELD FROM VIDEO MEETING FORM BY PYTHON

   models/jitsi.py

3. USE EXTERNAL API TO SEND PASSWORD TO JITSI-MEET BY JAVASCRIPT

   views/template.xml
   
4. CHECK SECURITY OPTIONS ICON IN JITSI-MEET

   LOOK AT GREEN ICON AT JITSI-MEET WEB PAGE
