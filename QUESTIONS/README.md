# Questions and Answers for Programming

This repository contains coding samples, algorithm implementations, and application examples across multiple programming languages and frameworks.

## 📁 Folder Structure

### ARTIFICIALINTELLIGENCE/

Currently empty. Reserved for future artificial intelligence and machine learning related projects and exercises.

### GOLANG/

Currently empty. Reserved for future Go programming language exercises and applications.

### PYTHON/

Python programming exercises focusing on algorithms, data structures, and automation scripts.

#### 📄 Arrays/
- **Description:** Array manipulation algorithms and problem-solving exercises
- **Content:**
  - Longest substring without repeating characters algorithm
  - Finding closest value in array using NumPy
  - Array operations and transformations

#### 📄 Pairs/
- **Description:** Advanced array algorithms for finding unique pairs with equal sums
- **Content:**
  - Algorithm to find all unique pairs in unsorted arrays with equal sum
  - Interactive testing and validation scripts
  - Unit tests
  - Detailed usage documentation
- **Key Features:**
  - Handles unsorted arrays efficiently
  - Eliminates duplicate pairs
  - Formatted output display

#### 📄 Search/
- **Description:** Docker and Kubernetes automation scripts
- **Subdirectory:** SEARCH_DOCKERFILES_KUBERNETES/
  - Python script to search GitHub repositories for Dockerfiles
  - Extracts Docker image names from FROM statements
  - Kubernetes deployment configuration for running the search script
  - Uses GitHub API for repository and code search

#### 📄 UndirectedGraph/
- **Description:** Graph theory implementations and traversal algorithms
- **Content:**
  - **ShortestPath/**: Breadth-First Search (BFS) algorithm for finding shortest paths
  - **Traversal/**: Depth-First Search (DFS) algorithm for graph traversal
- **Key Concepts:**
  - Undirected graph representation using adjacency lists
  - Queue-based BFS implementation
  - Stack-based DFS implementation

### REACT/

React JavaScript applications demonstrating various UI patterns and API integrations.

#### 📄 FACEBOOK/
- **Description:** React application with social media inspired layout and components
- **Content:**
  - Navigation bar with responsive design
  - List and item components
  - Map integration
  - Layout data management
  - Icon system

#### 📄 LAYOUT/
- **Description:** React application showcasing component layout patterns and organization
- **Content:**
  - Reusable layout components
  - Navigation container pattern
  - Item and list rendering patterns
  - Component composition examples

#### 📄 STOCKSDATA/
- **Description:** React application for fetching and displaying stock market data
- **Content:**
  - Date-based stock data search functionality
  - API integration with HackerRank JSON Mock API
  - Dynamic data rendering based on API responses
  - Error handling for no results scenarios
- **Key Features:**
  - Date input with specific format (d-mmmm-yyyy)
  - Display of Open, Close, High, and Low stock values
  - Conditional rendering based on data availability
  - Unit tests with Jest

## 🛠️ Technologies Used

- **Python 3.x**: NumPy, Requests, Regular Expressions
- **React**: Functional and class components, hooks, service workers
- **Docker & Kubernetes**: Container orchestration, ConfigMaps, Jobs
- **Testing**: Jest, Python unittest
- **API Integration**: GitHub API, Stock data APIs
